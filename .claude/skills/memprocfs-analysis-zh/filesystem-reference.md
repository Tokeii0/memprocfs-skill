
# MemProcFS 虚拟文件系统参考

本文档提供了 MemProcFS 公开的虚拟文件系统结构的参考。

## 根目录 (`/`)

-   **/conf/**: 配置文件和设置。
-   **/forensic/**: 用于取证分析的模块。
-   **/misc/**: 杂项实用程序和信息。
-   **/proc/**: 每个正在运行的进程的目录，由其 PID 标识。
-   **/sys/**: 系统范围的信息和模块。
-   **/vm/**: 与虚拟机相关的信息（如果适用）。

## 取证模块 (`/forensic/`)

-   **/csv/**: 各种模块以 CSV 格式的输出。
-   **/files/**: 从内存中恢复的文件。
-   **/findevil/**: FindEvil 恶意软件扫描程序的结果。
-   **/json/**: 各种模块以 JSON 格式的输出。
-   **/ntfs/**: NTFS 文件系统分析。
-   **/prefetch/**: 预取文件分析。
-   **/timeline/**: 系统活动时间线。
-   **/web/**: Web 浏览工件。
-   **/yara/**: YARA 扫描结果。

## 系统信息 (`/sys/`)

-   **/certificates/**: 系统证书信息。
-   **/drivers/**: 已加载的内核驱动程序。
-   **/memory/**: 物理内存映射。
-   **/net/**: 网络连接和统计信息。
-   **/objects/**: 内核对象。
-   **/pool/**: 内核池信息。
-   **/proc/**: 正在运行的进程的摘要。
-   **/services/**: 系统服务。
-   **/syscall/**: 系统调用信息。
-   **/sysinfo/**: 常规系统信息。
-   **/tasks/**: 计划任务。
-   **/users/**: 用户帐户和会话。

## 进程信息 (`/proc/<pid>/`)

-   **/bin/**: 进程可执行文件及其加载的模块。
-   **/console/**: 控制台命令历史记录。
-   **/files/**: 打开的文件句柄。
-   **/handles/**: 所有打开的句柄。
-   **/heaps/**: 进程堆信息。
-   **/memmap/**: 进程的虚拟内存映射。
-   **/minidump/**: 进程的小型转储。
-   **/modules/**: 已加载的模块 (DLL)。
-   **/threads/**: 进程线程。
-   **/token/**: 安全令牌信息。
-   **/vmemd/**: 作为单个文件的进程的整个虚拟内存。
